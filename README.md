## Installation

1. Clonez le dépôt :
    ```bash
    git clone https://github.com/SolalDFG/mon-site-web.git
    ```
2. Accédez au répertoire du projet :
    ```bash
    cd mon-site-web
    ```
3. Ouvrez le fichier `index.html` dans votre navigateur pour explorer le site localement.
